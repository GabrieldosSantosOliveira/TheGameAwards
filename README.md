## ✔️ Pré Requisitos
- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- [Node.js](https://nodejs.org/en/)

## 🚀 Como executar

Clone o repositório
```
git clone https://github.com/GabrieldosSantosOliveira/TheGameAwards.git
```
Instale as dependências
```
npm i
```
Execute com o expo
```
npm run start
```
